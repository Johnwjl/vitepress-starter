# Code Standards (Web FE)

## 注释

### Vue

- vue 文件中的 template 必须加注释，若文件较大添加 start end 注释
- vue 文件的 data, 非常见单词要加注释
- 为每个 `prop`添加注释
- vue 文件的 methods，每个 method 必须添加注释
- store 中的 state, mutation, action 等必须加注释

### API

- 为`API`添加注释
