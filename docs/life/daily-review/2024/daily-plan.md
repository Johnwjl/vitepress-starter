## 饮食计划

#### 早餐 BreakFast

- 碳水：全麦面包 / 杂粮粥 / 玉米 
- 蛋白质：鸡蛋 * 2
- 脂肪：坚果
- 维生素：香蕉
- 补剂类：蛋白粉、肌酸粉、男士复合维生素、鱼油

#### 工作日主餐 Dinner

- 主食：杂粮米 / 土豆泥 / 红薯 / 紫薯 
- 蛋白质：鸡蛋 * 0.5
- 脂肪：鸡胸 * 1
- 菜品：蒜蓉蚝油 西兰花 + 胡萝卜 / 青(1)红(1)椒+洋葱(1)+木耳(200g) 炒蛋(*2)
- 水果：小番茄 / 黄瓜

#### 工作日晚 汤

- 菠菜/西红柿/丝瓜 + （ 豆腐 + 榨菜 + 蛋花汤 ）
