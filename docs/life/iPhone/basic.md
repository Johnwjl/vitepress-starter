# iPhone📱使用指南
> 写给爸妈的苹果手机基本操作手册

## 介绍

- `iPhone`(苹果手机📱)，音同`哎凤`，是`Apple`(苹果公司) 音同`哎剖` 生产发行的手机。
- `iPhone`的手机系统，称作`ios` 音同`哎～欧～唉斯` 系统。

## 基本操作

- 将手机号码添加到通讯录

- 切换应用

- FaceTime 视频通话

### 短信
  - 置顶某条短信：在短信列表，长按该条短信，选择`置顶`
  - 批量删除短信：在短信列表，双指按住下滑
